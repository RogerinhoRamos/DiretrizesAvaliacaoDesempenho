Diretrizes de Avaliação

Guia Prático para Liderança na Usinagem

5 Competências Oficiais

1. Age como um especialista

Domínio técnico:
Conhece parâmetros, ferramentas, materiais
Resolução de problemas:
Identifica causa raiz, não só sintoma
Ensina outros:
Compartilha conhecimento, desenvolve colegas
Melhoria contínua:
Aplica lições aprendidas, otimiza processos
Critério de Notas
A
Resolve problemas complexos + ensina + inova processos
B
Domina sua função + ajuda colegas + aplica conhecimento
C
Conhece o básico + executa tarefas + busca ajuda quando precisa

2. Colabora competentemente e com integridade

Comunicação eficaz:
Informações claras entre turnos/setores
Trabalho em equipe:
Ajuda colegas sem prejudicar própria produção
Gestão de conflitos:
Resolve problemas sem escalações desnecessárias
Confiabilidade:
Cumpre acordos, é transparente
Critério de Notas
A
Líder natural + resolve conflitos + constrói relacionamentos
B
Colabora ativamente + comunica bem + é confiável
C
Trabalha em equipe + evita conflitos + cumpre básico

3. Foco nos negócios

Visão de custos:
Entende impacto financeiro das suas ações
Equilíbrio qualidade/prazo:
Não sacrifica um pelo outro
Orientação a resultados:
Foca no objetivo final, não só na tarefa
Sustentabilidade:
Considera recursos (energia, materiais, tempo)
Critério de Notas
A
Propõe melhorias que geram economia + visão estratégica
B
Equilibra custo/qualidade/prazo + entende o negócio
C
Cumpre metas + entende básico de custos + segue diretrizes

4. Profissionalismo

Organização:
Posto limpo, ferramentas no lugar, documentação em dia
Foco no cliente:
Pensa na aplicação final da peça
Métodos estruturados:
Segue procedimentos, tem sequência lógica
Padrão de qualidade:
Não aceita "mais ou menos"
Critério de Notas
A
Referência em organização + melhora processos + zero defeito
B
Sempre organizado + segue padrões + qualidade consistente
C
Organização básica + segue procedimentos + qualidade aceitável

5. Toma iniciativas e Age com foco

Antecipa problemas:
Age antes que pare a produção
Autonomia:
Decide dentro de sua competência sem depender sempre do líder
Proatividade:
Faz além do mínimo necessário
Inovação:
Testa métodos diferentes, busca eficiência
Critério de Notas
A
Antecipa tudo + alta autonomia + inova constantemente
B
Proativo + decide bem + busca melhorias
C
Age quando necessário + autonomia básica + cumpre função
